## ğŸ”„ Common Workflows

### New Account Setup
1. Create account on website
2. Open VaultMind Password Manager
3. Click "Add Password"
4. Enter title (e.g., "Netflix Account")
5. Click "Generate"
6. Copy generated password
7. Use on website
8. Return to VaultMind
9. Fill remaining fields
10. Save

### Password Change
1. Open website
2. Go to password change page
3. Open VaultMind
4. Find password entry
5. Click Edit
6. Generate new password
7. Copy new password
8. Paste on website
9. Save in VaultMind
10. Confirm on website

### Account Recovery
1. Open VaultMind
2. Search for account
3. View username/email
4. Check secure notes for security questions
5. Use password to login
6. Update password if needed
